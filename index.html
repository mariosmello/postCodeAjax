postCodeAjax - Demo

Plugin desenvolvido para auxiliar o cadastro de endereço, fazendo com que ao digitar o CEP (09351-020 ou 09351020) o sistema retorne e preencha o endereço completo, fazendo foco no campo de número.

Implementação básica Chame o plugin usando o ID do formulário como seletor e configure as classes de cada campo

$('#form1').postcode();
.postcode_ajax - Campo de CEP
.postcode_btn - Botão de Buscar (anula o envio do formulário mesmo apertando enter)
.street_ajax - Nome da rua
.area_ajax - Nome do bairro
.number_ajax - Número (apenas para foco)
.city_ajax - Nome da Cidade
.state_ajax - Nome do Estado (Sigla)
.postcode_loading - Loading exibido no momento da consulta

Implementação multipla É possível enviar implementar em 2 formularios diferentes com a mesma chamada, mantendo o padrão de classes nos campos

$('#form2,#form3').postcode();

Implementação personalizada Possibilidade de alteração das classes

$('#form4').postcode({
  btn: ".postcode_btn_custom",
  postcode: ".postcode_ajax_custom",
  street: ".street_ajax_custom",
  area: ".area_ajax_custom",
  city: ".city_ajax_custom",
  state: ".state_ajax_custom",
  number: ".number_ajax_custom",
  loading:'.postcode_loading_custom'
  hideSubmit: true // Oculta o botão de busca durante o loading
});
